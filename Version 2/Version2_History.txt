Date: 2026-04-21

FILE: src/store/sqlserver-store.js
FUNCTION: validateLogin() [Line 964-997]

CHANGES:
- Updated SQL query to use new schema
- Old: Queried dbo.users table
- New: Queries Login table with Employee join

SQL QUERY BEFORE:
  SELECT TOP 1 EmpID, EmpUsername, EmpFullName, EmpDesignation
  FROM dbo.users
  WHERE (EmpUsername = @username OR EmpEmail = @username)
  AND EmpPassword = @password

SQL QUERY AFTER:
  SELECT TOP 1 L.EmpID, L.Username, E.EmpFirstName, E.EmpLastName,
               E.EmpName, E.EmpDept
  FROM Login L
  INNER JOIN Employee E ON L.EmpID = E.EmpID
  WHERE L.Username = @username AND L.Password = @password AND L.IsActive = 1

REASON: Schema alignment with new TimeChime database design
